One-Step Denoising

The pretrained UNet predicts noise ε̂ in a single forward pass. Clean estimate: x̂₀ = (x_t − √(1−ᾱ_t)·ε̂) / √ᾱ_t. Results degrade significantly at higher t — too much information has been destroyed for a single step to recover.

1.4

Iterative Denoising

Strided timesteps 990→0 (stride 30, 34 steps total). Starting at i_start=10 (t=690). DDPM update rule: x_t' = (√ᾱ_t'·β_t)/(1−ᾱ_t)·x̂₀ + (√α_t·(1−ᾱ_t'))/(1−ᾱ_t)·x_t + v_σ, where learned variance v_σ is added at each step.

Classifier-Free Guidance (CFG)

CFG formula: ε = εᵤ + γ(εc − εᵤ) with γ=7. Both runs use the same prompt "a high quality photo". CFG dramatically sharpens results by amplifying the direction the conditional score differs from the unconditional one.

Visual Anagrams

At each denoising step: ε₁ = CFG(x_t, p₁), ε₂ = flip(CFG(flip(x_t), p₂)), ε = (ε₁+ε₂)/2. Averaging forces the image to simultaneously satisfy both prompts — one upright, one when rotated 180°.

1.9

Hybrid Images

Factorized Diffusion: at each step, compute two CFG noise estimates ε₁ (p₁) and ε₂ (p₂), then combine: ε = lowpass(ε₁) + highpass(ε₂) using a Gaussian blur (kernel=33, σ=2) as the lowpass filter. The result contains low-frequency structure from p₁ and high-frequency detail from p₂ — so the perceived subject changes with viewing distance.

Part B

Flow Matching from Scratch

Training a UNet for single-step denoising and iterative flow matching on MNIST, including time-conditioning and class-conditioning with classifier-free guidance.

B · 1.1

Implementing the UNet

Custom UNet with the following atomic blocks:

  • Conv — Conv2d(k=3,s=1,p=1) → BatchNorm2d → GELU. Preserves spatial dims, changes channels.
  • DownConv — Conv2d(k=3,s=2,p=1) → BN → GELU. Downsamples by 2.
  • UpConv — ConvTranspose2d(k=4,s=2,p=1) → BN → GELU. Upsamples by 2.
  • Flatten — AvgPool2d(7) collapses 7×7 → 1×1.
  • Unflatten — ConvTranspose2d(k=7,s=1,p=0) expands 1×1 → 7×7.
  • Concat — Channel-wise skip connections via torch.cat().

Composed into ConvBlock (Conv→Conv), DownBlock (ConvBlock→DownConv), and UpBlock (UpConv→ConvBlock). Hidden dim D=128 for Part 1, D=64 for Part 2.

Analysis: With MSE loss on pure Gaussian noise inputs, the model learns E[x|z] — the conditional expectation given the noise. Since z carries no digit-specific information, this expectation collapses to the mean across all training images: a blurry, averaged blend of all MNIST digits. The model converges to the centroid of the training distribution — ghostly, indistinct digit-like shapes — rather than learning to generate sharp, specific digits. This is the fundamental limitation of one-step MSE-based generation.
B · 2.1–2.3

Time-Conditioned UNet & Flow Matching

Flow matching interpolates between noise x₀ ∼ N(0,I) and data x₁: x_t = (1−t)x₀ + tx₁. The target flow (velocity) is u = x₁−x₀. We train: L = E‖(x₁−x₀) − u_θ(x_t,t)‖².

Time t ∈ [0,1] is injected via two FCBlocks (Linear→GELU→Linear→GELU): unflatten = unflatten × t₁, up1 = up1 × t₂. Sampling uses Euler integration: x ← x + u_θ(x,t)·dt for 50 steps.

Hyperparameters: D=64, lr=1e-2, batch=64, 10 epochs, ExponentialLR scheduler (γ=0.1^(1/10)).

B · 2.4–2.6

Class-Conditioned UNet with CFG

Class conditioning adds two more FCBlocks for one-hot class vector c ∈ ℝ¹⁰. Modulation: unflatten = c₁·unflatten + t₁, up1 = c₂·up1 + t₂. During training, condition is dropped with probability p_uncond=0.1 (set c=0) to enable classifier-free guidance.

At sampling: u_cfg = u_uncond + γ(u_cond − u_uncond) with γ=5. 300 Euler steps during training, 50 at inference.

B · Bells & Whistles

Training Without LR Scheduler

The exponential scheduler (γ=0.1^(1/10)) decays lr from 1e-2 to ~1e-3 over 10 epochs, preventing late-stage oscillations. To compensate for its removal, we use a constant lr=1e-3 from the start — Adam's adaptive per-parameter rates then provide implicit stabilization comparable to the scheduled run.
